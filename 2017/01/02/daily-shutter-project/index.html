Daily Shutter Project

Jan 2, 2017 | Chitchat | 阅读

中文：每日快门计划。

我这人没有写日记或周记的习惯。16年11月底的时候，突然回想了一下自己一整年来经历过什么，却发现自己能有印象的只有那么少少的几件，这让我有一种虚度光阴的感觉，非常不爽。我总觉得，人生中，每天或多或少都有那么些幸福时光、或是小确幸、或是郁闷时刻、或是尴尬时刻等等。这些点点滴滴，都是构成我们人生回忆的碎片，在将来记起时，都会让我们脸上浮现静谧的笑容。可我却记不得多少。

我记得以前看过一个TED演讲（忘了演讲的题目了），演讲者为了记下自己和家人的美好时光，每天录制1秒的视频。对，没看错，就是每天1秒的视频。这样1年可以拼接成一个6分多钟的视频，演讲者觉得这份视频非常的有意义，这是他和家人珍贵的回忆。我也觉得这非常的有意义。

于是，结合自己的心情和这个TED演讲给我的启发，我给自己启动了这个Daily Shutter Project，中文名“每日快门”。在这个project里，我将坚持每天拍至少1张照片，写至少一句话并记录下来。这样一年下来就有至少365个瞬间和至少365句话了。用这个有趣的project来代替日记，将我自己的一整年串连起来。这也算是我2017年的一个小目标或者说是挑战吧，毕竟要每天坚持下来还是不容易的。

这个project已经从2016年11月30日开始实施，到今天第34天，执行状况良好。目前没有找到一个很好的方式或App进行记录，所以暂时先用Day One做记录，说不好就这样一直用这个App记录也说不定，除非找到更好的记录方式。

希望2017年结束，我有一个完整的Daily Shutter Project的执行结果。
